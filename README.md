#### Maze Generator

Generates mazes with a single solution (no loops) using a randomized recursive algorithm.
Controls: Solve mazes with WASD, each input traverses the maze until reaching a junction or dead end and highlights every square it traversed. Whenever reaching the end of the maze, it instantly generates a new one.
Time tracking: Your individual time to solve for every previous maze, your average time to solve, and your total time spent solving is displayed below the maze.
